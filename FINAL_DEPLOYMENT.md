## ⚙️ 技术栈总览

### 前端技术
- **框架**：Next.js 14 (App Router)
- **UI库**：React 18 + TypeScript
- **样式**：Tailwind CSS + Radix UI
- **图标**：Lucide Icons
- **状态**：React Hooks
- **认证**：NextAuth.js

### 后端技术
- **API**：Next.js API Routes
- **数据库**：SQLite (dev) / PostgreSQL (prod)
- **ORM**：Prisma
- **文件存储**：Base64 (本地)

### 设计系统
- **风格**：Apple Human Interface
- **配色**：Apple蓝 #007AFF
- **字体**：SF Pro Display / PingFang SC
- **动画**：150-300ms，Spring缓动
- **间距**：8px栅格系统
